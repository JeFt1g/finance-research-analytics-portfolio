# Formulas

Forecast revenue = `prior revenue * (1 + growth)`

Operating income = `revenue * operating margin`

NOPAT = `operating income * (1 - tax rate)`

Free cash flow = `NOPAT - reinvestment`

Terminal value = `final FCF * (1 + terminal growth) / (discount rate - terminal growth)`

Enterprise value = `PV forecast FCF + PV terminal value`

Equity value = `enterprise value - net debt`

Implied share price = `equity value / shares outstanding`

EV/Revenue = `enterprise value / revenue`

EV/EBITDA = `enterprise value / EBITDA`

P/E = `market cap / net income`
