# Project Plan

## Objective

Create a DCF and comparables valuation project with sensitivity analysis, charts, and final recommendation.

## Inputs

Historical financials, comparables, and scenario assumptions.

## Calculations

Revenue forecast, operating income, NOPAT, FCF, PV of FCF, terminal value, enterprise value, equity value, implied share price, and peer multiples.

## Outputs

DCF forecast, comparable valuation, sensitivity table, five charts, and report.

## Step-by-Step Build Plan

1. Load inputs.
2. Build base DCF.
3. Build bull/base/bear cases.
4. Calculate peer multiples.
5. Build sensitivity table.
6. Generate charts and report.

## Subagents Responsible

Project Architect, Finance Analyst, Quant Developer, Data Visualization, Report Writer, and QA Agent.

## Acceptance Criteria

`src/main.py` runs, generates five charts, and writes the required valuation report.

## Future Improvements

Add real filings, dilution, and detailed operating schedules.
