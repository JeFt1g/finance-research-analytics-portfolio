# Python Portfolio Analysis Case Study

## Problem

Evaluate a sample multi-asset portfolio by return, volatility, drawdown, allocation, and correlation.

## Data Used

- `data/sample_portfolio_prices.csv`
- `data/sample_portfolio_weights.csv`

## Return Analysis

The project calculates daily returns, cumulative returns, weighted portfolio returns, and portfolio value.

## Risk Analysis

The project calculates annualized volatility, Sharpe ratio, max drawdown, asset risk-return metrics, and a correlation matrix.

## Charts

Charts include portfolio value, cumulative returns, asset allocation, risk-return positioning, and correlation heatmap.

## Real-World Use

This structure can support investment policy reviews, client portfolio summaries, and benchmark comparison once real market data is added.

## Limitations

Sample data is fake, short, and excludes fees, taxes, dividends, rebalancing, and benchmark tracking error.
