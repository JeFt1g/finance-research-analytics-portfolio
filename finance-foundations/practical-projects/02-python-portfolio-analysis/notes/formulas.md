# Formulas

Daily return = `(price today / price yesterday) - 1`

Portfolio return = sum of `asset return * asset weight`

Cumulative return = cumulative product of `(1 + return) - 1`

Annualized return = `(1 + total return) ^ (252 / daily observations) - 1`

Annualized volatility = `daily return standard deviation * sqrt(252)`

Sharpe ratio = `(return - risk-free rate) / volatility`

Max drawdown = largest percentage decline from a previous wealth peak.
