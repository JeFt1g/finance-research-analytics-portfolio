# Project Plan

## Objective

Build a company financial statement analysis project with ratios, trend charts, insights, risks, and a final report.

## Inputs

Annual revenue, gross profit, operating income, net income, assets, liabilities, equity, and debt.

## Calculations

Year-over-year growth, gross margin, operating margin, net margin, ROE, current ratio, debt-to-equity, and asset turnover.

## Outputs

Ratio table, five charts, and a markdown report.

## Step-by-Step Build Plan

1. Load and validate data.
2. Calculate ratios and growth metrics.
3. Generate charts.
4. Export report.

## Subagents Responsible

Project Architect, Finance Analyst, Quant Developer, Data Visualization, Report Writer, and QA Agent.

## Acceptance Criteria

Script runs locally, generates five charts, and produces the required report.

## Future Improvements

Add real data adapters and peer benchmarking.
