# Mispricing Risk Dashboard

## One-Sentence Summary

CLI dashboard prototype for mispricing, liquidity, short interest, and arbitrage failure risk.

## Why It Matters

This project turns Limits to Arbitrage from a note into a runnable tool.

## Core Idea

The starter version uses sample data, beginner-readable functions, and no paid APIs. It is intentionally small so the formula and assumptions remain visible.
