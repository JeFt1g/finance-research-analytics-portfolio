# CAPM Beta Calculator

## One-Sentence Summary

Calculator for beta, CAPM expected return, and alpha.

## Why It Matters

This project turns CAPM from a note into a runnable tool.

## Core Idea

The starter version uses sample data, beginner-readable functions, and no paid APIs. It is intentionally small so the formula and assumptions remain visible.

## Framework

Framework: implement the smallest useful CAPM calculation, show sample input, then explain limitations.

## Simple Example

Run `python main.py` from this folder to see the default sample output.
