# Option Pricing Engine

## One-Sentence Summary

Black-Scholes pricing engine for calls, puts, Greeks, and implied volatility.

## Why It Matters

This project turns Black-Scholes from a note into a runnable tool.

## Core Idea

The starter version uses sample data, beginner-readable functions, and no paid APIs. It is intentionally small so the formula and assumptions remain visible.

## Key Terms

## Framework

Framework: implement the smallest useful Black-Scholes calculation, show sample input, then explain limitations.

## Simple Example

Run `python main.py` from this folder to see the default sample output.

## Real-World Use

Use it as a base for a dashboard, API, test suite, or AI-agent tool.
