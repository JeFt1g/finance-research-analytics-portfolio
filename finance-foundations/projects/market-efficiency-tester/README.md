# Market Efficiency Tester

## One-Sentence Summary

Event-study framework for before/after stock returns and benchmark comparison.

## Why It Matters

This project turns Market Efficiency from a note into a runnable tool.

## Core Idea

The starter version uses sample data, beginner-readable functions, and no paid APIs. It is intentionally small so the formula and assumptions remain visible.
