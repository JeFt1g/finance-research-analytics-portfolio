# CAPM Sharpe - Simple Explanation

## One-Sentence Summary

CAPM links expected return to systematic market risk measured by beta.

## Why It Matters

It is a default tool for estimating cost of equity, comparing performance, and separating market exposure from stock-specific return.

## Core Idea

CAPM says investors should be compensated for bearing market risk, not for risks that can be diversified away. Beta measures how sensitive an asset is to the market portfolio. Higher beta implies higher expected return if the equity risk premium is positive.

## Key Terms

- Risk-free rate
- Market return
- Equity risk premium
- Beta
- Systematic risk
- Idiosyncratic risk
- Alpha
- Security Market Line

## Formula / Model

```txt
Expected Return = Risk-Free Rate + Beta*(Market Return - Risk-Free Rate)
Beta = covariance(stock, market) / variance(market)
Alpha = Actual Return - CAPM Expected Return
```

## Simple Example

With a 4% risk-free rate, 9% expected market return, and beta of 1.2, expected return is 10%.

## Real-World Use

DCF valuation, hurdle rates, performance attribution, stock risk screens, and cost of equity estimates.

## Limitations

- The true market portfolio is hard to observe.
- Beta changes through time.
- One factor can miss size, value, momentum, rates, credit, and other exposures.
