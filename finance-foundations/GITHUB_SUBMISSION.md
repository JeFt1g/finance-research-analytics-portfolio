# GitHub Submission Guide

## What To Publish

Publish the repository containing this `finance-foundations/` folder. The GitHub landing page should point reviewers to:

- `finance-foundations/README.md`
- `finance-foundations/PORTFOLIO.md`
- `finance-foundations/PROJECT_INDEX.md`
- `finance-foundations/QA_REPORT.md`
- dashboard screenshots under `outputs/screenshots/`

Finance research and analytics portfolio with theory notes, article processing, Python finance apps, Streamlit dashboards, charts, reports, and screenshots.
